# Super Signal Monitor 📡

A comprehensive C++ application that monitors both WiFi and cellular signal strength across multiple platforms. Provides real-time monitoring, data logging, and visualization capabilities.

## Features

- **Dual Monitoring**: Simultaneous WiFi and cellular signal monitoring
- **Cross-Platform**: Works on Windows and Linux
- **Real-time Display**: Live signal strength with quality ratings
- **Data Logging**: CSV and SQLite database support
- **Visualization**: KML export for Google Earth mapping
- **Professional UI**: Clean terminal interface with auto-refresh

## Supported Signal Types

### WiFi
- Signal strength (RSSI) in dBm
- Network names (SSID)
- MAC addresses
- Frequency bands

### Cellular
- GSM/3G/LTE/5G signals
- Multiple technology support
- Cell tower information
- Operator data
